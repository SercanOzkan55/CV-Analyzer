# CV Analyzer

CV Analyzer is a FastAPI + React based platform for CV/job-description matching, ATS analysis, and recommendation generation.

## Tech Stack

- Backend: FastAPI, Python
- Frontend: React, Vite
- Database: PostgreSQL (with pgvector support)

## Features

- CV upload and analysis
- Semantic and keyword matching
- Skill coverage and missing skill insights
- ATS compatibility scoring and suggestions
- Multilingual analysis output support
